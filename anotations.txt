Comandos do docker

-> Subir container:
    docker compose up -d

-> Abrir terminal:
    docker compose exec -it redis redis-cli

Comandos do redis:

SET: ALterar, criar UMA chave + seu valor
GET: Cponsultar uma chave  e seu valor

M ---> Many
MSET: Cadastrar ou alterar multiplas chaves
MGET : Consultar Multiplas chaves

SETNX: Crie se não existir
STRLEN: Verifica o tamanho de um string
GETRANGE: Seleciona um intervalo de texto
    Ex: SET nome: "ABC DEF"
        GETRANGE: nome 0(star) 2(end)
        "AB"
SETRANGE: Altera os caracteres com base no novo valor passado + sua posição

INCRBY : Soma valores numeriocs
INCRBYFLOAT: Soma valores numeriocs quando os dados soa FLOAT
DECRBY: Retira valores

EXPIRE: Define um tempo de expiração para a chave
PERSIST: Cancela o tempo de expiração e tona a chave "permanente" novamente
PEXPIRE: Define o tempo de expiração em milisegundos

SCAN: Retorna as chaves criadas (uma espécie de MGET para nomes de chaves)

LPUSH: Criar a adicionar itens à esquerda
RPUSH: Adiciona itens à direita da lista (Amobos os comandos podem criar listas)
LRANGE: Verifica o tamnaho da lista
LLEN: Verificar tamanho da lista
LINDEX: Busca por itens com base no índice
LSET: Alterir item na lista 
LINSERT: Insere item na lista com base em uma elemento. OBS: Ele só aceita texto em AFTER|BEFORE
LMOVE: Mover itens entre lista
